# components/

Reusable React components go here, organized by feature.

## Suggested structure

```
components/
  auth/       ← Components used on the auth/login page
  game/       ← Wordle board, tiles, keyboard, etc.
  chat/       ← AI chatbot UI components
  ui/         ← Generic shared components (Button, Input, Modal, etc.)
```

Each component should be its own file, e.g. `game/WordleBoard.js`.
